To deploy this project, you can simply fork it, clone it and set the Enviroment Variables inside a dotenv file.

Required on the dotenv file:

```

    RIOT_TOKEN={YOUR RIOT API TOKEN}
    PREFIX={YOUR PREFIX}
```
